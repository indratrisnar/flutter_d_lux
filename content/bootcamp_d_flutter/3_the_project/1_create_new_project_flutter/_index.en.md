If it's a terminal, go to the storage directory where you want to save the project, then run the command `flutter create â€“template app -projectname`. If using Android Studio there is a Flutter project menu option. However, this bootcamp uses Visual Studio Code so the sequence of steps is described below.

1. Open the command palette

{{< image src="3_application_type.png" caption="" alt="alter-text" height="" width="" position="left" command="fill" option="q100" class="img-fluid" title=""  webp="false" >}}
